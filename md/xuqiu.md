任务二：开发线上图书系统（单页面应用 SPA）
某书店老板希望上线网上卖书平台，你需开发一个前端应用页面。
________________________________________
主页面需显示：
•	书名
•	价格
•	分类
•	删除按钮
•	顶部必须有 添加书籍 按钮
________________________________________
功能要求：
1. 新增书籍 Popup
需要弹窗输入：
•	书名
•	价格
•	分类
•	描述
提交后需加入书籍列表。
________________________________________
2. 点击书籍 → 编辑书籍 Popup
弹窗需允许：
•	修改书名、价格、分类、描述
•	保存后更新主页面
•	提供删除按钮以移除该书籍
________________________________________
3. 无需后端
所有书籍数据保存在 前端数组 中。
________________________________________
加分项（可选）：
•	checkbox 批量删除
•	Sort By 排序功能
•	部署至 GitHub Pages
•	任何能展示额外实力的功能
________________________________________
技术要求：
•	必须使用 TypeScript
•	必须使用 Redux
•	评分重点：
o	功能完整性
o	代码结构
o	可维护性
o	效率
