# 在线图书管理系统 (React SPA)

这是一个纯前端的单页面应用，展示现代 React 技术栈的工程化实践。

## 技术栈
- **核心框架**：React + TypeScript + Vite
- **状态管理**：Redux Toolkit
- **样式/UI**：Tailwind CSS

## 快速启动
```bash
npm install
npm run dev
```

## 核心功能与亮点
- **图书 CRUD**：所有数据闭环在前端 Redux 内存中。
- **功能特性**：支持搜索、多字段排序、批量选中删除。
- **架构设计**：采用 **Feature-Driven Design** (按功能模块划分目录)，高内聚低耦合。
